# Violin Locker
The Violin Locker is an LP locking smart-contract system to be deployed on EVM compatible chains. It is provided as-is.

## Deploy
```
yarn deploy avax
```

### Environment variables
- PRIVATE_KEY
- ETHERSCAN_APIKEY
